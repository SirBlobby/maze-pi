REST API

POST /api/run

Start running a block program. The robot stops any currently running program or Python code before starting.

Request Body

{
  "commands": [
    { "action": "forward",    "duration": 2, "speed": 80 },
    { "action": "turn_left",  "duration": 1, "speed": 80 },
    { "action": "set_led",    "color": "green" },
    { "action": "beep",       "duration": 0.3 }
  ]
}

Response

{ "status": "started", "count": 4 }
POST /api/stop

Stop the robot immediately. Halts both block programs and Python code execution.

Response

{ "status": "stopped" }
GET /api/status

Returns the current state of the block program executor.

Response

{
  "running": true,
  "current": { "action": "forward" }
}
GET /api/sensor/sonar

Returns the current sonar (ultrasonic) distance reading in centimetres.

Response

{ "sensor": "sonar", "value": 23.4, "mock": false }
GET /api/sensor/line

Returns the state of all four infrared line sensors. Each value is true when a black line is detected under that sensor (left to right: S1–S4).

Response

{ "sensor": "line", "value": [false, true, true, false], "mock": false }
GET /api/system

Returns system statistics for the robot's Raspberry Pi.

Response

{
  "battery_v":   7.8,
  "battery_pct": 65,
  "cpu_pct":     12,
  "cpu_temp_c":  52,
  "ram":         { "percent": 38, "used_mb": 310 },
  "disk":        { "percent": 21, "used_gb": 3.1, "total_gb": 15.0 },
  "ip":          "192.168.1.42",
  "mock":        false
}
GET /api/info

Returns whether the server is running in mock mode (no physical robot).

Response

{ "mock": false, "hardware_available": true }
POST /api/run_code

Execute a Python program with access to all robot functions. The program runs in a background thread; poll /api/code/status for output. Any running block program is stopped first.

Request Body

{ "code": "forward(2, 80)\nturn_left(1, 80)\nprint('done')" }

Response

{ "status": "started" }
GET /api/code/status

Returns the execution state and buffered output of the Python code runner. Poll this after calling /api/run_code.

Response

{
  "running": false,
  "output":  ["Distance: 42.1", "Turning...", "[stopped]"],
  "error":   null
}

When a runtime error occurs, error contains the exception message and running is false.

{
  "running": false,
  "output":  [],
  "error":   "NameError: name 'oops' is not defined"
}
GET /api/saves

List all saved block programs.

Response

{ "saves": ["maze run 1", "test turn"] }

Load a save — GET /api/saves/<name>

{ "xml": "<xml>...</xml>", "commands": [...] }

Write a save — PUT /api/saves/<name>

Body: any JSON object (the full workspace state). Returns {"status":"saved","name":"..."}.

Delete a save — DELETE /api/saves/<name>

Returns {"status":"deleted","name":"..."}.

Python Code API

The following functions are available in the Code editor. All movement functions are blocking — they drive the robot and return only after the duration has elapsed.
Speed ranges from 50–100 (default 80). Duration is in seconds.

Movement

forward(duration=1, speed=80)

Drive the robot forward, then stop.

ParameterTypeDefaultDescription
durationfloat1Seconds to drive
speedint80Speed 50–100

Example

forward(2, 80)
forward(0.5)
backward(duration=1, speed=80)

Drive the robot backward, then stop.

Example

backward(1, 70)
turn_left(duration=1, speed=80)

Spin left in place for the given duration. Higher speed = sharper turn (speed 50 → turn rate 1.0, speed 100 → turn rate 2.0).

Example

turn_left(0.8, 100)
turn_right(duration=1, speed=80)

Spin right in place for the given duration.

Example

turn_right(0.8, 100)
strafe_left(duration=1, speed=80)

Slide sideways to the left using mecanum wheels.

strafe_right(duration=1, speed=80)

Slide sideways to the right using mecanum wheels.

stop()

Stop all motors immediately. Does not block.

Control

wait(duration=1)

Pause the program. Responds to the Stop button unlike time.sleep().

Example

forward(1, 80)
wait(0.5)
backward(1, 80)
print(*args)

Print values to the output terminal on the Code page. Works like the built-in print().

Example

d = distance_cm()
print('Distance:', d, 'cm')

sensors = read_line_sensor()
print('Line sensors:', sensors)

Sensors

distance_cm()

Read the ultrasonic sonar sensor. Returns a float (centimetres) or None if unavailable.

Example

while True:
    d = distance_cm()
    print('Distance:', d)
    if d is not None and d < 20:
        stop()
        set_led('red')
        break
    wait(0.1)
read_line_sensor()

Read all four infrared line sensors. Returns a list of 4 booleans — True when a black line is detected. Sensors are ordered left to right: S1, S2, S3, S4.

Example

s = read_line_sensor()
print(s)

if s[1] or s[2]:
    print('Line under center sensors')
follow_line(duration=5, speed=60)

Autonomously follow a black line on the floor for the given duration. Uses all four infrared sensors to steer.

Example

set_led('blue')
follow_line(10, 60)
set_led('off')
print('Done following line')

Effects

set_led(color='off')

Set the robot's RGB LEDs. Valid colors: red orange yellow green blue purple white off.

Example

set_led('green')
forward(2, 80)
set_led('red')
stop()
wait(1)
set_led('off')
beep(duration=0.2)

Sound the piezo buzzer for the given duration in seconds.

Example

beep(0.1)
wait(0.1)
beep(0.1)

Full Example

print('Maze runner starting')
set_led('blue')

for i in range(3):
    d = distance_cm()
    print(f'Loop {i+1}, distance: {d} cm')

    if d is not None and d < 25:
        set_led('red')
        beep(0.2)
        turn_right(0.8, 100)
    else:
        set_led('green')
        forward(1.5, 80)

set_led('off')
print('Done')
